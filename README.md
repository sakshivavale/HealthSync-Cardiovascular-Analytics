# 🩺 HealthSync – Predictive Analysis of Lifestyle Habits on Cardiovascular Health

HealthSync is a Data Analytics and Machine Learning project focused on predicting cardiovascular risk using lifestyle and behavioral factors. The project analyzes habits such as sleep, exercise, stress, smoking, and diet to identify hidden patterns related to heart health.

The system uses Exploratory Data Analysis (EDA), statistical testing, SMOTE-based data balancing, and Random Forest Classification to generate real-time cardiovascular risk predictions through an interactive Streamlit dashboard.

## 📂 Dataset Information

The dataset contains **8,763 patient records** with medical and lifestyle-related attributes such as:

- Age
- Cholesterol
- Blood Pressure
- Exercise Hours
- Sleep Duration
- Smoking Habits
- Stress Levels
- Diet
- Diabetes

---

## 📈 Key Insights

- Individual lifestyle factors showed weak linear correlations with heart attack risk.
- Combined lifestyle habits revealed strong non-linear relationships.
- Cholesterol, Age, and Exercise Hours were among the most important predictive features.
- Feature Isolation improved real-world prediction reliability.
